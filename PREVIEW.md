## 📱 手机预览方法

### 方法一：本地网络访问（推荐）
1. 确保手机和电脑在同一个WiFi网络
2. 在电脑上启动HTTP服务器：
   ```bash
   cd "D:\WorkBuddy\LuojiStudio\working\ongoing\aiedu-plus"
   python -m http.server 8080
   ```
3. 获取电脑的本地IP地址：
   - Windows: 在命令提示符输入 `ipconfig`
   - 查找"无线局域网适配器 WLAN"或"以太网适配器"的IPv4地址
4. 在手机浏览器访问：`http://[电脑IP]:8080`

### 方法二：使用在线工具
1. 将整个 `aiedu-plus` 文件夹上传到Vercel、Netlify或GitHub Pages
2. 通过生成的URL在手机上访问

### 方法三：直接文件传输
1. 将 `index.html`、`resources.html`、`cases.html`、`style.css` 发送到手机
2. 在手机文件管理器中打开 `index.html`
