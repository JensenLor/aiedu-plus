# 学习资料填写说明（WPS 专用版）

以后你主要打开这一个文件录入资料：

- **主录入入口：`data/resources.csv`**

只有当你要新增一个全新的大类时，再去改：

- `data/categories.csv`

---

## 1. 你平时主要改哪个文件？

### `data/resources.csv`
这是你以后最常用的录入表。

你只需要在 WPS 里：
1. 打开 `resources.csv`
2. 找到底部已经铺好的分类模板行
3. 复制最接近的一行
4. 把内容改成你的新资料
5. 把“启用”改成 `1`
6. 保存

页面就会自动读取它。

---

## 2. resources.csv 中文表头说明

| 表头 | 作用 |
| --- | --- |
| `资源ID` | 资源唯一标识，建议用英文、数字、连字符，例如 `tinkercad` |
| `分类ID` | 资源属于哪个分类，例如 `3d`、`ai`、`programming` |
| `标题` | 页面上显示的名称 |
| `网址` | 资源链接 |
| `一句话简介` | 卡片上的简短介绍 |
| `标签` | 多个标签用 `|` 分隔，例如 `免费|在线|入门` |
| `详细说明` | 点开“查看详情”后显示的说明 |
| `补充信息` | 补充标签，多个内容用 `|` 分隔 |
| `图标` | 建议直接填 emoji，例如 `🧊` |
| `角标` | 右上角标记，例如“推荐”“官方”“免费” |
| `启用` | `1` 表示显示，`0` 表示隐藏 |
| `排序` | 数字越小越靠前 |

---

## 3. 现在已经预铺好的模板分类

`resources.csv` 底部已经放好了这些隐藏模板行：

- AI 与机器学习（`ai`）
- 编程入门（`programming`）
- 硬件与创客（`hardware`）
- 3D 建模与打印（`3d`）
- 机器人与自动化（`robotics`）
- 游戏开发（`games`）
- 数学与算法（`math`）
- 实用工具（`tools`）

这些模板默认都是：
- `启用 = 0`，所以不会显示到网页上
- 内容只是示范提示语，你可以直接覆盖
- 改完后把 `启用` 改成 `1`

这套表现在已经是“打开就能录”的状态。

---

## 4. 如果要新增分类

再打开 `data/categories.csv`。

它的中文表头含义如下：

| 表头 | 作用 |
| --- | --- |
| `分类ID` | 分类唯一标识，例如 `3d` |
| `分类名称` | 页面显示名称 |
| `图标` | 分类图标 |
| `分类说明` | 当前分类的说明文字 |
| `排序` | 越小越靠前 |
| `启用` | `1` 显示，`0` 隐藏 |

同样也保留了一行隐藏模板，可直接复制使用。

---

## 5. WPS 使用注意

- 当前两个 CSV 都已保存为 **UTF-8 with BOM**，WPS 打开一般不会乱码
- 如果你另存文件，优先保持 `UTF-8` 或 `CSV UTF-8`
- 如果某个单元格里有逗号，建议使用英文双引号包起来
- 标签或补充信息建议用 `|` 分隔
- `分类ID` 必须对应 `categories.csv` 里的某个分类
