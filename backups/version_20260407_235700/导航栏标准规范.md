# 导航栏标准规范

**版本**：v1.0
**创建时间**：2026年4月7日 23:22
**最后更新时间**：2026年4月7日 23:22
**适用页面**：所有HTML页面（index.html、resources.html、works.html等）

---

## 1. 核心原则

### 1.1 统一性原则
- 所有页面的导航栏必须**完全一致**
- 包括：结构、尺寸、样式、图标、文字、间距
- 任何修改必须应用到所有页面

### 1.2 一致性检查清单
- [ ] 结构位置一致
- [ ] 尺寸大小一致
- [ ] 图标引用一致
- [ ] 样式类名一致
- [ ] 响应式行为一致

---

## 2. 结构标准

### 2.1 HTML结构
```html
<body>
    <div class="container">
        <!-- 导航栏 -->
        <nav>
            <div class="nav-brand">
                <img src="images/罗辑智造社二次元头部形象.png" alt="Logo" class="nav-logo">
                <span class="nav-title">罗辑智造社</span>
            </div>
            <ul>
                <li><a href="index.html" class="active">🏠 首页</a></li>
                <li><a href="resources.html">🎓 学习资源</a></li>
                <li><a href="works.html">MakerVerse</a></li>
            </ul>
        </nav>
        
        <!-- 页面内容 -->
        ...
    </div>
</body>
```

### 2.2 结构要求
1. **容器位置**：`<nav>`必须在`<div class="container">`内部
2. **品牌区域**：使用`<div class="nav-brand">`包裹logo和标题
3. **菜单列表**：使用`<ul>`和`<li>`结构
4. **链接格式**：使用`<a>`标签，包含emoji图标和文字
5. **激活状态**：当前页面链接添加`class="active"`

## 3. 尺寸标准

### 3.1 整体尺寸
| 组件 | 宽度 | 高度 | 说明 |
|------|------|------|------|
| 导航栏整体 | 100% | 自动 | 自适应容器宽度 |
| Logo图片 | 50px | 50px | 固定尺寸 |
| 品牌文字 | 自动 | 自动 | 跟随字体大小 |
| 菜单项 | 自动 | 40px | 包含内边距 |

### 3.2 间距标准
| 间距类型 | 数值 | 说明 |
|----------|------|------|
| Logo与文字间距 | 10px | 水平间距 |
| 品牌区与菜单间距 | 20px | 水平间距 |
| 菜单项间距 | 15px | 水平间距 |
| 导航栏上下内边距 | 15px | 垂直内边距 |
| 导航栏左右内边距 | 20px | 水平内边距 |

### 3.3 字体尺寸
| 元素 | 字体大小 | 字体粗细 | 说明 |
|------|----------|----------|------|
| 品牌标题 | 20px | 700 | 粗体 |
| 菜单文字 | 16px | 600 | 半粗体 |
| 菜单图标 | 18px | 正常 | emoji图标 |

## 4. 图标标准

### 4.1 Logo图标
| 属性 | 值 | 说明 |
|------|----|------|
| 文件路径 | `images/罗辑智造社二次元头部形象.png` | **绝对路径** |
| 文件名 | 罗辑智造社二次元头部形象.png | 中文文件名 |
| 文件类型 | PNG | 透明背景 |
| CSS类名 | `nav-logo` | 必须使用 |
| alt属性 | `Logo` | 英文描述 |
| 尺寸 | 50px × 50px | 固定尺寸 |
| 圆角 | 0 | 直角 |
| 边框 | none | 无边框 |
| 阴影 | none | 无阴影 |
| object-fit | contain | 保持比例 |

### 4.2 菜单图标（Emoji）
| 菜单项 | Emoji | 代码 | 说明 |
|--------|-------|------|------|
| 首页 | 🏠 | U+1F3E0 | 房屋图标 |
| 学习资源 | 🎓 | U+1F393 | 学士帽图标 |
| MakerVerse | (无) | - | 纯文字 |

**重要**：
1. Emoji直接写在HTML中，不使用图片
2. 保持emoji与文字之间的一个空格
3. 使用统一的emoji版本

## 5. 样式标准

### 5.1 CSS类名
必须使用以下类名：
```css
/* 容器类名 */
.container     /* 页面主容器 */
nav            /* 导航栏标签 */
.nav-brand     /* 品牌区域 */
.nav-logo      /* Logo图片 */
.nav-title     /* 品牌文字 */
nav ul         /* 菜单列表 */
nav li         /* 菜单项 */
nav a          /* 菜单链接 */
nav a.active   /* 当前页面链接 */
```

### 5.2 颜色标准
使用CSS变量定义：
```css
:root {
    --primary-color: #FF6B35;    /* 主色：橙色 */
    --secondary-color: #FF8C42;  /* 副色：浅橙色 */
    --border-color: #333;        /* 边框色：深灰色 */
    --text-dark: #333;           /* 深色文字 */
    --text-medium: #666;         /* 中等色文字 */
    --text-light: #999;          /* 浅色文字 */
    --card-bg: #FFF9F5;          /* 卡片背景：浅橙色 */
    --shadow-color: #FF8C42;     /* 阴影色：浅橙色 */
}
```

### 5.3 导航栏样式
从`style.css`继承以下样式：
```css
/* 导航栏基础样式 */
nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 15px 20px;
    background: white;
    border-bottom: 3px solid var(--border-color);
    margin-bottom: 30px;
}

/* 品牌区域样式 */
.nav-brand {
    display: flex;
    align-items: center;
    gap: 10px;
}

/* Logo样式 */
.nav-logo {
    width: 50px;
    height: 50px;
    border-radius: 0;
    border: none;
    box-shadow: none;
    object-fit: contain;
}

/* 品牌文字样式 */
.nav-title {
    font-size: 20px;
    font-weight: 700;
    color: var(--text-dark);
}

/* 菜单列表样式 */
nav ul {
    display: flex;
    list-style: none;
    gap: 15px;
    margin: 0;
    padding: 0;
}

/* 菜单链接样式 */
nav a {
    padding: 10px 15px;
    text-decoration: none;
    color: var(--text-dark);
    font-size: 16px;
    font-weight: 600;
    border-radius: 8px;
    transition: all 0.25s ease;
    border: 2px solid transparent;
}

/* 悬停状态 */
nav a:hover {
    background: #FFF7ED;
    color: var(--primary-color);
    border-color: rgba(230, 126, 34, 0.2);
}

/* 激活状态 */
nav a.active {
    background: #FFF3E0;
    color: var(--text-dark);
    border-color: var(--secondary-color);
}
```

## 6. 响应式标准

### 6.1 断点设置
```css
/* 平板设备 (768px以下) */
@media (max-width: 768px) {
    nav {
        flex-direction: column;
        gap: 15px;
        padding: 15px;
    }
    
    .nav-brand {
        justify-content: center;
    }
    
    nav ul {
        flex-wrap: wrap;
        justify-content: center;
    }
}

/* 手机设备 (480px以下) */
@media (max-width: 480px) {
    .nav-title {
        font-size: 18px;
    }
    
    nav a {
        padding: 8px 12px;
        font-size: 14px;
    }
}
```

### 6.2 响应式要求
1. **平板**：导航栏改为垂直布局，菜单居中
2. **手机**：减小字体和间距，保持可点击性
3. **触屏优化**：确保点击区域足够大（最小44px）

### 7.1 视觉验证
1. 打开所有页面，逐个检查导航栏
2. 比较尺寸、颜色、间距是否一致
3. 检查响应式布局是否正常

## 8. 修改流程

### 8.1 修改前
1. 备份当前版本：`backup_pages.bat`
2. 阅读本规范文档
3. 确定修改范围

### 8.2 修改中
1. 修改`style.css`中的通用样式
2. 同步修改所有HTML页面的导航栏
3. 保持结构、类名、属性一致

### 8.3 修改后
1. 运行验证方法检查一致性
2. 创建新备份记录修改
3. 更新本规范文档（如有必要）

## 9. 常见错误

### 9.1 已发生的错误
1. **结构不一致**：导航栏在container内外位置不同
2. **尺寸不一致**：自定义样式覆盖通用样式
3. **图标不一致**：使用不同的logo图片路径
4. **类名不一致**：使用自定义类名而非标准类名

### 9.2 避免方法
1. 始终从`style.css`继承样式
2. 始终使用标准HTML结构
3. 始终使用标准CSS类名
4. 修改前先备份，修改后验证
