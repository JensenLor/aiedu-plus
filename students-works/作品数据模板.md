# 作品数据填写模板

## 文件位置
- **作品基本信息**：填写到 `作品信息.csv` 中
- **作品详细内容**：存放在 `作品详细内容/` 文件夹中
- **作品文件**：存放在 `downloads/` 文件夹中（ZIP包）

---

## 1. 在 `作品信息.csv` 中填写

### CSV格式说明
```
作品ID,学生学号,作品名称,作品描述,技术标签,难度等级,创建日期,预览摘要(100字内),下载文件
```

### 填写示例
```
work005,ZZS0002,智能家居控制系统,基于Arduino的智能家居控制系统，支持语音控制和手机App,"Arduino,Python,物联网,语音识别",困难,2026/3/20,基于Arduino的智能家居控制系统，支持语音控制和手机App远程操控，包含灯光、温湿度、安防监控功能。,work005_智能家居控制系统.zip
```

### 字段说明
- **作品ID**：`work` + 三位数字，如 `work005`, `work006`
- **学生学号**：从 `学生基本信息.csv` 中选择，如 `ZZS0001`, `ZZS0002`
- **作品名称**：简洁明了的作品名称
- **作品描述**：一句话描述作品功能
- **技术标签**：用英文逗号分隔，如 `"Arduino,Python,物联网"`
- **难度等级**：简单/中等/困难
- **创建日期**：`YYYY/MM/DD` 格式
- **预览摘要**：100字内的简短介绍
- **下载文件**：ZIP包文件名，如 `work005_智能家居控制系统.zip`

## 2. 创建作品详细内容

### 文件夹结构
```
作品详细内容/
├── work005/                    # 作品ID作为文件夹名
│   ├── 01-封面图片.jpg         # 作品封面图片
│   ├── 02-项目截图1.jpg        # 项目截图
│   ├── 03-项目截图2.jpg        # 项目截图
│   ├── 04-项目截图3.jpg        # 项目截图
│   └── 详细说明.md             # 详细说明文档
└── work006/
    ├── 01-封面图片.jpg
    ├── 详细说明.md
    └── ...
```

### 详细说明.md 模板
```markdown
# [作品名称]

## 作品简介
[这里写作品的详细简介，200-300字]

## 技术特点
- [技术特点1]
- [技术特点2]
- [技术特点3]

## 实现过程
1. [第一步：...]
2. [第二步：...]
3. [第三步：...]

## 技术难点与解决方案
- **难点1**：[描述难点]
  - 解决方案：[如何解决的]
- **难点2**：[描述难点]
  - 解决方案：[如何解决的]

## 学习收获
- [学到了什么1]
- [学到了什么2]
- [学到了什么3]

## 作品文件说明
- `main.py` - 主程序文件
- `config.json` - 配置文件
- `README.md` - 使用说明
- [其他文件...]

## 使用方法
1. [第一步使用方法]
2. [第二步使用方法]
3. [第三步使用方法]

## 扩展建议
- [可以改进的地方1]
- [可以改进的地方2]
```

## 3. 创建作品ZIP包

### 位置
`downloads/` 文件夹中

### 命名规则
`[作品ID]_[作品名称].zip`

示例：`work005_智能家居控制系统.zip`

---

## 4. 快速开始指南

### 第一步：填写作品信息
1. 打开 `作品信息.csv`
2. 在最后一行添加新作品信息
3. 保存文件

### 第二步：创建作品文件夹
1. 在 `作品详细内容/` 中创建新文件夹，以作品ID命名
2. 添加封面图片和截图
3. 创建 `详细说明.md` 并填写内容

### 第三步：上传作品文件
1. 将作品源代码/文件打包成ZIP
2. 放到 `downloads/` 文件夹中
3. 确保文件名与CSV中的"下载文件"字段一致

### 第四步：更新网站数据
运行 `create_zip.bat` 或手动运行 `update_website.py` 更新JSON数据
