# 🤖 微信快速助手 - 免填表格系统

## 🎯 核心功能
**你说话/发文字 → 我自动填表格 → 网站立即更新**

## 📝 使用方式

### 方式1：直接发给我处理（推荐）
直接把学生或作品信息发给我，我会帮你更新所有表格。

**学生信息格式（任选一种）：**
```
新增学生：张三，初中二年级，擅长Python和Web开发，喜欢AI项目，2026年4月7日加入
```
```
学生：李四，小学五年级，Scratch很厉害，爱做游戏
```
```
添加学生：王五，高中一年级，硬件达人，机器人比赛获奖
```

**作品信息格式：**
```
新增作品：学生zzy001，作品名"AI绘画助手"，用Python和深度学习，描述：基于Stable Diffusion的绘画工具
```
```
作品：学生stu004，作品"智能家居控制"，Arduino+Python，难度中等
```

### 方式2：运行脚本
在文件夹中双击 `voice_interactive.bat` 进入交互模式。

### 方式3：命令行
```bash
python voice_to_table.py "新增学生：张三，初中二年级"
```

## 🔧 自动处理内容

### 学生信息自动提取：
- ✅ **姓名**：自动识别
- ✅ **年级**：小学/初中/高中 + 年级
- ✅ **擅长标签**：Python、HTML/CSS、JavaScript、AI应用、硬件、Scratch等
- ✅ **分类**：ai、web、hardware、game（自动根据擅长标签分类）
- ✅ **个人简介**：从描述中提取
- ✅ **加入日期**：自动识别或使用今天日期
- ✅ **学号**：自动生成（stu001, stu002...）

### 作品信息自动提取：
- ✅ **作品名称**：自动识别
- ✅ **学生学号**：自动关联
- ✅ **技术标签**：Python、TensorFlow、Arduino、HTML等
- ✅ **难度等级**：简单/中等/困难
- ✅ **作品描述**：从输入中提取
- ✅ **预览摘要**：自动生成（前100字）
- ✅ **作品ID**：自动生成（work001, work002...）
- ✅ **ZIP文件**：自动创建并关联

## 📁 系统自动完成的工作

1. **更新3个CSV文件**：
   - 学生基本信息.csv
   - 作品信息.csv  
   - 社交数据.csv

2. **创建文件夹结构**：
   ```
   students/
   ├── stu001/          # 学生文件夹
   │   ├── work001/     # 作品文件夹
   │   └── work002/
   ```

3. **生成ZIP下载包**：
   ```
   downloads/
   ├── work001_作品名称.zip
   └── work002_作品名称.zip
   ```

4. **更新网站数据**：
   - 自动运行 `simple_update_data.py`
   - 生成最新的 `students.json`
   - 网站立即显示新内容
