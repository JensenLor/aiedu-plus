## 常见测试问题

### 问题1：学生卡片未显示
- 检查Excel表格中的学号是否与学生文件夹名称匹配
- 确认Excel文件保存为.xlsx格式
- 检查Python脚本是否成功运行

### 问题2：预览摘要不显示
- 确认作品文件夹内有preview.txt文件
- 检查preview.txt内容不超过100字
- 确认文件编码为UTF-8

### 问题3：图片未显示
- 确认图片文件在正确的位置
- 检查图片格式是否为常见格式（PNG, JPG等）
- 确认文件权限可读
