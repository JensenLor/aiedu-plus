作品名称：
深海智者・小海 | 免费海豚伙伴・本地智慧引擎

预览摘要（100 字内）：
一款海洋主题的本地智能聊天网页应用，以海豚小海为形象，无需 API、免登录免付费，支持文字 / 语音交互，能陪聊、解答问题、记用户偏好，视觉上有动态海洋特效，体验生动有趣。

技术栈：
KIMI、DEEPSEEK、本地数据处理

作品简介：
这是一款主打 “本地智能、完全免费” 的聊天类网页应用，以拟人化的海豚 “小海” 为交互形象，融合海洋主题视觉设计（动态海浪、气泡、游动的海洋生物），无需依赖外部 API 接口、无需登录或付费，即可实现自然的人机对话。应用聚焦轻量化、趣味性与易用性，适配移动端与桌面端，为用户提供无门槛的陪伴式聊天体验。

功能详情：
1、核心聊天交互：支持文字输入（自适应高度输入框）、回车发送，消息区分用户 / AI 侧，带时间戳与动画效果；
2、本地智能回复：内置多场景回复逻辑，覆盖问候、感谢、算术、笑话、海洋科普、心情陪伴等，可记忆用户姓名 / 爱好；
3、语音交互：集成 Web Speech API，支持语音输入（麦克风识别）、语音播报（可开关），播报时头像有动态特效；
4、视觉动效：动态海浪、气泡生成、海洋生物游动动画，聊天框悬浮效果、输入 / 加载动效，整体海洋风格沉浸式体验；
5、响应式设计：适配移动端与桌面端，界面布局自动调整，保证不同设备的交互体验；
6、交互细节：输入框自适应高度、滚动条美化、发送按钮反馈、加载状态提示（打字动效）等。
