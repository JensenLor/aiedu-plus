## 一、HTML 结构

所有页面必须使用以下标准结构：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题 - 罗辑智造社</title>
    <link rel="stylesheet" href="style.css">
    <style>
        body {
            padding-top: 20px;
        }
    </style>
</head>
<body>
    <!-- 导航栏 -->
    <nav>
        <div class="nav-brand">
            <img src="images/罗辑智造社二次元头部形象.png" alt="Logo" class="nav-logo">
            <span class="nav-title">罗辑智造社</span>
        </div>
        <ul>
            <li><a href="index.html">🏠 首页</a></li>
            <li><a href="resources.html">🎓 学习资源</a></li>
            <li><a href="works.html">MakerVerse</a></li>
        </ul>
    </nav>

    <!-- 内容区域 -->
    <div class="container">
        <!-- 页面内容 -->
    </div>

    <!-- 页脚 -->
    <footer>
        <p>© 2026 罗辑智造社 - 中小学STEM/创客教育平台</p>
        <p>培养学生 6 大 AI 能力，让创意成为现实</p>
        <p>地址：中国 | 邮箱：contact@aiedu.plus</p>
    </footer>
</body>
</html>
```

---

## 二、必须引用 style.css

**必须**在 `<head>` 中引用统一的 `style.css`：
```html
<link rel="stylesheet" href="style.css">
```

**禁止**在页面内联 `<style>` 中覆盖以下样式：
- `.nav-logo` — Logo 尺寸、圆角、边框
- `.nav-title` — 标题颜色、大小
- `nav` — 导航栏整体样式
- `.nav-brand` — 品牌区域布局

从 `style.css` 中定义的精确尺寸：

| 元素 | 属性 | 值 |
|------|------|-----|
| **nav** | padding | `15px 30px` |
| **nav** | margin-bottom | `30px` |
| **nav** | border-radius | `20px` |
| **nav** | border | `3px solid var(--border-color)` |
| **nav-logo** | width | `45px` |
| **nav-logo** | height | `45px` |
| **nav** | max-width | `1200px` |
| **nav** | margin-left/right | `auto` (居中) |
| **nav-logo** | border-radius | `0` (方形/直接显示原图) |
| **nav-logo** | border | `none` |
| **nav-logo** | box-shadow | `none` |
| **nav-title** | font-size | `24px` |
| **nav-title** | color | `var(--primary-color)` (#FF6B35) |

## 六、新页面创建检查清单

创建新页面时，必须确认：

- [ ] HTML 中包含 `<link rel="stylesheet" href="style.css">`
- [ ] 导航栏 HTML 结构与上面标准结构完全一致
- [ ] 没有内联 `<style>` 覆盖 `.nav-logo`、`.nav-title`、`nav` 等样式
- [ ] Logo 图片使用 `images/罗辑智造社二次元头部形象.png`
- [ ] 内容包裹在 `<div class="container">` 中
- [ ] 包含标准 `<footer>` 区域
- [ ] **body 添加 `padding-top: 20px`**（确保导航栏顶部有留白，与其他页面一致）

## 七、常见错误及解决方案

### 错误1：Logo 变成方形而非圆形
**原因**：页面内联 `<style>` 中覆盖了 `.nav-logo` 的 `border-radius: 50%`  
**解决**：删除该覆盖样式，使用 `style.css` 的默认样式

### 错误2：导航栏高度异常
**原因**：自定义了 `nav` 的 padding 或 height  
**解决**：删除自定义样式，使用 `style.css` 的 `padding: 15px 30px`

### 错误3：Logo 尺寸不对
**原因**：设置了 `.nav-logo` 的 `width`/`height` 不是 45px  
**解决**：保持 45px × 45px 的标准尺寸
